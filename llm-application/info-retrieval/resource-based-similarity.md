# 语言学资源驱动的相似度计算

#### 第 4 篇：语言学资源驱动的相似度计算

**标题建议：** _同义词词林与知网 Hownet：中文相似度计算的语言学路径_

**内容：**

* **引子**：除了统计和神经网络，还有一条路——人工构建的语言学知识库
* **同义词词林（CiLin）**
  * 词林的层次编码体系（大/中/小类 → 词群 → 原子词群）
  * 相似度计算方法：基于编码树的层级距离
  * 优势：不需要任何训练，直接查表
  * 局限：覆盖率有限，无法处理新词
* **知网 Hownet 义原匹配**
  * 什么是"义原"（Sememe）：语义的最小单位
  * 知网的知识表示：概念 → 义原树
  * 基于义原的词语相似度计算（刘群方法 / 基于路径长度）
  * 义原层面的距离如何聚合为词语层面的相似度
* **语言学方法 vs 统计方法 vs 深度学习方法的三角对比**
  * 可解释性、泛化能力、计算成本各维度分析
  * 混合使用的可能性
